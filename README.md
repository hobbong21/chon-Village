### 7. 노드 확장 시스템 (NEW! 🔥🔥🔥)
가족 관계를 인증받은 사용자만 학교, 직장, 동호회 등 다양한 노드(그룹)를 생성하고 참여할 수 있는 시스템

#### 7.1 노드 타입
- **학교** (school): 교육 기관, 동문회
- **직장** (company): 회사, 조직
- **동호회** (club): 취미, 관심사 모임
- **커뮤니티** (community): 지역, 온라인 커뮤니티
- **팬클럽** (fanclub): 연예인, 스포츠 팬클럽

#### 7.2 노드 관리 기능
- **노드 생성**: 타입 선택, 이름, 설명, 공개 설정
- **노드 검색**: 타입별, 공개 설정별 필터링
- **노드 가입**: 공개 노드 자동 가입 또는 승인 대기
- **노드 초대**: 레벨 3 이상 멤버가 다른 사용자 초대
- **멤버 관리**: 멤버 목록 조회 (레벨 2 이상)

#### 7.3 역할 기반 시스템
각 노드마다 고유한 역할 정의:
- **레벨 3 (소유자)**: 노드 생성자, 최고 관리 권한
- **레벨 2 (관리자)**: 멤버 관리, 초대 권한
- **레벨 1 (회원)**: 기본 멤버

#### 7.4 5단계 관계 레벨 시스템
활동과 인증을 통한 자동 레벨 업그레이드:

| 레벨 | 이름 | 조건 | 권한 |
|------|------|------|------|
| **5** | 🏆 공인 | 인증 10회 + 활동점수 500점 | 노드 생성, 타인 인증, 모든 정보 공유 |
| **4** | ⭐ 리더 | 인증 5회 + 활동점수 200점 | 멤버 초대/관리, 전체 프로필 공유 |
| **3** | 🔥 활동 | 활동점수 100점 | 대부분 정보 공유 |
| **2** | 🔗 연결 | 인증 1회 | 기본 정보 공유 |
| **1** | ⏳ 대기 | 초기 상태 | 정보 공유 없음 |

#### 7.5 활동 점수 시스템
사용자 참여를 장려하는 포인트 시스템:
- 노드 가입: +10점
- 타인 인증: +20점
- 게시글 작성: +5점
- 댓글 작성: +2점
- 좋아요: +1점
- 멤버 초대: +15점
- 이벤트 참여: +50점

#### 7.6 상호 인증 시스템
- 동일 노드 멤버끼리 상호 인증 가능 (레벨 2 이상)
- 인증받을 때마다 인증 횟수 +1, 활동점수 +20
- 인증 제공자도 활동점수 +5
- 일정 인증 횟수 달성 시 레벨 자동 업그레이드

#### 7.7 초대 시스템
- 레벨 3 이상 멤버만 초대 가능
- 초대 토큰 자동 생성 (7일 유효)
- 초대 수락 시 즉시 레벨 2로 시작
- 초대자는 활동점수 +15

#### 7.8 프로필 공개 설정
관계 레벨에 따른 자동 정보 공개 제어:
- **레벨 1**: 정보 공유 없음
- **레벨 2**: 기본 정보 (이름, 헤드라인)
- **레벨 3**: 확장 정보 (경력, 학력)
- **레벨 4**: 전체 프로필 (연락처 제외)
- **레벨 5**: 모든 정보 (연락처 포함)

#### 7.9 노드 시각화 (예정)
- 노드 간 연결 네트워크 시각화
- 사용자 중심 멀티 노드 관계도
- 노드 통계 대시보드

### 8. 가족 관계 시스템 (UPDATED! ⭐)
- **가족 구성원 관리**:
  - 기본 단위: 아버지, 어머니, 나
  - 추가 가능: 형제/자매, 배우자, 자녀
  - 필수 입력: 이름(한글/영문) + 연락처 중 최소 2개
  - 연락처 누적 관리 (기존 정보 유지, 신규 추가)
  
- **관계 인증 시스템**:
  - 상호 인증을 통한 관계 확인
  - 인증된 구성원끼리만 프로필 정보 공개
  - 인증 요청/수락 기능
  
- **프로필 사이드바 미니 네트워크** (UPDATED! ⭐):
  - 좌측 프로필 카드 하단에 컴팩트 가족 네트워크 표시
  - 300px 높이의 미니 D3.js 시각화
  - 가족 통계 표시 (총 인원, 인증 완료, 인증 대기)
  - 빠른 액션 버튼 (가족 추가, 전체 보기)
  - 노드 클릭으로 빠른 정보 확인
  
- **전체 네트워크 시각화** (D3.js):
  - 인터랙티브 노드 기반 가계도
  - 관계별 색상 구분 (아버지: 파랑, 어머니: 분홍, 본인: 초록, 배우자: 보라, 형제/자녀: 주황/노랑)
  - 관계 유형별 링크 (부모-자식, 부부, 형제)
  - 인증 상태 표시 (점선: 미인증, 실선: 인증)
  - 노드 드래그 & 드롭
  - 줌/팬 기능
  - 노드 클릭 시 상세정보 패널 표시

## 📝 기능별 API 엔드포인트

### 인증 (NEW! 🎉)
- `POST /api/auth/login` - 로그인
- `POST /api/auth/register` - 회원가입

### 사용자 관리
- `GET /api/users` - 전체 사용자 목록 조회
- `GET /api/users/:id` - 특정 사용자 상세 조회
- `GET /api/users/:id/experiences` - 사용자 경력 조회
- `GET /api/users/:id/education` - 사용자 학력 조회
- `GET /api/users/:id/skills` - 사용자 스킬 조회
- `GET /api/users/:id/posts` - 사용자 게시물 조회
- `GET /api/users/:id/connections` - 사용자 연결 목록

### 게시물 관리
- `GET /api/posts` - 전체 게시물 피드 조회
- `POST /api/posts` - 새 게시물 작성
- `POST /api/posts/:id/like` - 게시물 좋아요
- `GET /api/posts/:id/comments` - 게시물 댓글 조회
- `POST /api/posts/:id/comments` - 댓글 작성

### 네트워크 관리
- `POST /api/connections` - 연결 요청 보내기
- `PUT /api/connections/:id` - 연결 수락/거부

### 가족 관계 관리 (NEW! 🔥)
- `GET /api/family/tree` - 가족 트리 조회 (인증된 구성원만)
- `POST /api/family/members` - 가족 구성원 추가
- `GET /api/family/members/:id` - 구성원 상세정보 조회
- `POST /api/family/verify/request` - 관계 인증 요청
- `POST /api/family/verify/accept` - 관계 인증 수락

### 노드 확장 시스템 (NEW! 🔥🔥🔥)
#### 노드 관리
- `GET /api/nodes` - 전체 노드 목록 (타입/공개 설정 필터링)
- `GET /api/nodes/:id` - 노드 상세 정보
- `POST /api/nodes` - 새 노드 생성
- `GET /api/nodes/:id/members` - 노드 멤버 목록 (레벨 2 이상)
- `GET /api/my-nodes` - 내가 가입한 노드 목록
- `GET /api/my-level` - 내 관계 레벨 현황

#### 멤버십 관리
- `POST /api/nodes/:id/join` - 노드 가입 신청
- `POST /api/nodes/:id/invite` - 멤버 초대 (레벨 3 이상)
- `POST /api/invitations/:token/accept` - 초대 수락
- `PUT /api/memberships/:id/verify` - 멤버 인증 (상호 인증)

## 💻 기술 스택

- **HTTP 클라이언트**: Axios
- **아이콘**: Font Awesome

## 🛠️ 로컬 개발

### 설치
```bash
npm install
```

### 데이터베이스 마이그레이션
```bash
# 로컬 D1 데이터베이스 마이그레이션
npm run db:migrate:local

# 테스트 데이터 삽입
npm run db:seed

# 데이터베이스 초기화 (리셋)
npm run db:reset
```

### 개발 서버 실행
```bash
# 빌드
npm run build

# PM2로 서버 시작
pm2 start ecosystem.config.cjs

# 서버 상태 확인
pm2 list

# 로그 확인
pm2 logs webapp --nostream

# 서버 재시작
npm run clean-port && pm2 restart webapp

# 서버 종료
pm2 delete webapp
```

## 📚 사용자 가이드

### 0. 로그인/회원가입 (NEW! 🎉)
1. **로그인**: `/login` 페이지에서 이메일과 비밀번호 입력
   - 데모 계정: john.doe@example.com (비밀번호는 임의)
2. **회원가입**: `/register` 페이지에서 필수 정보 입력
   - 이름, 이메일, 비밀번호, 한 줄 소개
   - 약관 동의 체크
3. **로그아웃**: 상단 메뉴의 로그아웃 버튼 클릭

### 1. 홈 피드
- 메인 페이지에서 모든 사용자의 게시물을 볼 수 있습니다
- 상단 텍스트 영역에 내용을 입력하고 "게시" 버튼을 클릭하여 게시물을 작성합니다
- 각 게시물에서 좋아요, 댓글, 공유 버튼을 사용할 수 있습니다
- 댓글 버튼을 클릭하면 댓글을 보고 작성할 수 있습니다

### 2. 네트워크
- 상단 메뉴에서 "네트워크"를 클릭합니다
- 다른 사용자 목록을 볼 수 있습니다
- "연결" 버튼을 클릭하여 연결 요청을 보냅니다
- 검색창에서 사용자를 검색할 수 있습니다 (향후 구현 예정)

### 3. 가족 관계도 (NEW! 🔥)

- 300px 크기의 컴팩트한 D3.js 네트워크 차트로 가족 구성원을 확인할 수 있습니다
- 노드를 클릭하면 간단한 정보를 팝업으로 볼 수 있습니다
- 통계 섹션에서 등록된 가족, 인증 완료, 인증 대기 수를 확인할 수 있습니다

- 전통 문양 테두리와 한지 느낌의 배경
- 촌수 표시 및 한국어 관계 명칭

- 🔴 어머니: 분홍색

- "가족 추가" 버튼을 클릭하여 새 구성원 등록

- 최소 2가지 식별 정보 필수 (이름 + 연락처)
- 연락처는 누적 관리 (삭제되지 않고 추가됨)

  - 소개
  - 경력 사항
  - 학력 사항
  - 스킬
  - 활동 (게시물)
